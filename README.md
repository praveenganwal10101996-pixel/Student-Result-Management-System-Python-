# Student Result Management System (Python)

##  Project Description
This is a Python-based Student Result Management System that allows users to:
- Enter student details
- Add marks for different subjects
- Automatically calculate total, percentage, and result
- Store results of different classes in separate Excel sheets

Each class has its own sheet inside a single Excel file.

---

##  Technologies Used
- Python
- openpyxl (for Excel file handling)
- Microsoft Excel

---

##  Features
- Add multiple students' results
- Dynamic subject-wise marks input
- Automatic total, percentage, and pass/fail calculation
- Separate Excel sheet for each class
- User-friendly input through terminal
- Data saved permanently in `.xlsx` format
